# Security Policy

This repository contains markdown skill files, examples, and documentation. It does not include executable runtime code with attack surface, secrets, credentials, or live data feeds.

If you find content that fabricates sanctions designations, vessel names, IMO numbers, citations, or other claims that could mislead a real compliance, AML, sanctions, or operational decision, please open an issue with the file path and the specific concern.

For sensitive disclosures, contact the maintainer via the GitHub profile listed in repository metadata.
